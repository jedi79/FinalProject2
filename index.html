I am an artist.

Primarily a artist of computer graphics, but I shape words as well as pixels. I have created this site to offer a gateway to my perspective. Sharp, simplistic, yet innovative, I have a lot to offer the current world of media. Enjoy.

I've been doing multimedia design for over 10 years, and have developed a good eye for it. I started doing graphic design while attending Northeastern University, where I started the old-fashioned way using a T-square, paper, and a drafting table. This helped me gain a good sense of shape, contrast, and space before I started working with raster graphics using Adobe Photoshop, and vector graphics using Adobe Illustrator. Luckily these skills came in handy when I began creating graphics for the web. At Emerson College, I learned how to create interactive web pages with HTML and CSS, and began working with photography, 3D animation, motion graphics, and digital video.

Back then, web designers didn't need to know much development code. So I've updated my skills. I'm on my way to becoming a true multimedia warrior.
